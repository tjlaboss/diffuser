diffusion.matrices module¶

Functions for dealing with the diffusion matrices

diffusion.matrices.gauss_seidel(L, U, S, x, k)¶

A Gauss-Seidel iterative solver

Theoretically, this is a more efficient solution because we only do math on the matrix elements that we know contain the right solution, and x is updated as we go.

In practice, this is slower than scipy’S builtin matrix solution methods (including inversion) because they use C or Fortran libraries.

Parameters

  • L (np.ndarray(ndim=2)) – Lower-triangular square matrix, containing the diagonal

  • U (np.ndarray(ndim=2)) – Upper-triangular square matrix, only off-diagnal terms

  • S (np.ndarray(ndim=1)) – Source vector

  • x (np.ndarray;) – Flux vector of last iteration

  • k (float) – Eigenvalue from the last iteration

Return type

np.ndarray

Returns

Flux solution vector
