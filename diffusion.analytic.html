class diffusion.analytic.AnalyticProblem1D(fill, width=None)¶

Bases: object

Base class for analytic solutions to homogeneous 1D problems

Parameters

  • fill (diffusion.material.Material) – Material the medium is filled with

  • width (float, cm) – Width or diameter of the medium [Default: None]

Variables

  • bg2 (float) – Geometric buckling (exactly 0 if infinite)

  • ngroups (int) – Number of energy groups

  • mag1 (float) – Relative magnitude of fast flux. Only present if `ngroups’ is 2.

  • mag2 (float) – Relative magnitude of thermal flux. Only present if `ngroups’ is 2.

  • ratio (float) – Ratio of mag1/mag2.
