add_region(fill, xwidth, num_nodes)¶

Add a material region to the problem

Adds a region of a single material and one or more nodes to the problem. Nodes are added from left to right.

Parameters

  • fill (diffusion.material.Material;) – Material with which to fill the new node(s).

  • xwidth (float; cm) – Width of the new region.

  • num_nodes (int) – Number of nodes to split the new region into.

run(SolverClass, kguess=None, plot_level=0)¶

Run the numerical simulation

Guide to plot levels:

Level 0: (default) no plots Level 1: Flux and fission source plots only Level 2: Level 1 + matrix spy() plots Level 3: Level 2 + L2norm convergence plot

Parameters

  • SolverClass (derived class of diffusion.solvers.BaseSolver) – Which numerical solver to use for the matrix solution

  • kguess (float; optional) – Guess to use for k_eff [Default: k_inf of self.fuel if available; otherwise 1]

  • plot_level (int; optional) – How much plotting to do [Default: 0]
